About CEAgui


CEAgui is a graphical-user-interface for CEA2, version 2 of NASA Glenn's computer program Chemical Equilibrium with Applications.   It facilitates making an input file for any of the CEA2 applications, calling the executable, and preparing the output files for viewing and downloading.


CEA/CEA2

CEA and CEA2 are very similar FORTRAN codes.   The analysis, definitions, and examples are the same as those given in the CEA manuals NASA RP-1311 Part I, 1994, and NASA RP-1311 Part II, 1996.   Some additional information is available at the website http://www.lerc.nasa.gov/WWW/CEAWeb/.   These programs require libraries of standard state thermodynamic and thermal transport properties of individual species.   CEAgui includes the current NASA Glenn libraries.


THERMODYNAMIC PROPERTIES

The thermodynamic data library for individual species in a readable formatted form is in a file called thermo.inp.   These data are documented in NASA TP-211556, 2002.    CEA2 uses a processed unformatted form of these data in a file called thermo.lib.   A convenient tool called ThermoBuild at http://cea.grc.nasa.gov/ generates tables of thermodynamic functions for any user-supplied temperature schedule.   ThermoBuild also includes a feature for downloading a subset of thermo.inp for use in other computer codes.   Equations and references are available at the website - see About ThermoBuild .


THERMAL TRANSPORT PROPERTIES

CEAgui's thermal transport properties are an updated version of the data documented in NASA TM-4647, 1995.   The readable formatted file is called trans.inp while the unformatted form used by CEA2 is called trans.lib.
